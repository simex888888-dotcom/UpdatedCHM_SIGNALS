БЫСТРЫЙ СТАРТ:
──────────────
1. Открой config.py и вставь токен бота:
   TELEGRAM_TOKEN = "7412589631:AAHxyz..."
   (получи токен у @BotFather в Telegram)

2. Установи зависимости:
   pip3 install -r requirements.txt

3. Запусти бота:
   python3 bot.py

4. Открой Telegram, найди своего бота и напиши /start

ФАЙЛЫ:
──────
bot.py           — главный файл (запускать его)
config.py        — настройки (только токен)
fetcher.py       — получение данных с OKX
indicator.py     — логика CHM BREAKER
scanner_multi.py — сканер для всех пользователей
handlers.py      — обработчики кнопок Telegram
keyboards.py     — кнопки меню
user_manager.py  — хранение настроек пользователей
users.json       — создастся автоматически

КОМАНДЫ БОТА:
─────────────
/start — открыть меню
/menu  — открыть меню
/stop  — остановить сигналы

КАЖДЫЙ ПОЛЬЗОВАТЕЛЬ НАСТРАИВАЕТ:
─────────────────────────────────
• Таймфрейм (1m / 5m / 15m / 30m / 1h / 4h / 1d)
• Интервал сканирования
• Фильтры (RSI / объём / паттерны / HTF тренд)
• Минимальное качество сигнала (1-5 звёзд)
• Цели TP1 / TP2 / TP3 (R:R)
• Минимальный объём монеты
